 ## Configurations
 
 **.gitconfig**
 ```.gitconfig
[alias]
  pm = pull origin master
  s = status -s
  p = pull
  ps = push
  b = branch
  curr = branch --show-current
  c = checkout
  cm = commit -m
  ca = commit -am
  ru = remote update
  asc = branch --sort=-committerdate
```

**_vimrc**
```
syntax on
set ignorecase
set smartindent
set nobomb
set tabstop=2
set shiftwidth=2
set expandtab
:imap jj <Esc>

fun! TrimWS()
  let l:save = winsaveview()
  keeppatterns %s/\s\+$//e
  call winrestview(l:save)
endfun

command! trimws call TrimWS()
```
